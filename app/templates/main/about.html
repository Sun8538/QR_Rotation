About the System

Smart Attendance Management System

The traditional attendance process used in many educational institutions is slow, error-prone, and vulnerable to proxy attendance, leading to inaccurate records. To overcome these issues, this project introduces a Smart Attendance Management System that leverages real-time cloud synchronization and dynamic QR code rotation.

Key Features

  • Dynamic QR codes that rotate every 30 seconds
  • Real-time attendance tracking and synchronization
  • Prevention of proxy attendance through device authentication
  • Comprehensive attendance analytics and reports
  • Session management for professors
  • Student enrollment and class management
  • Notifications for students and professors

Technology Stack

  • Python Flask for backend
  • SQLite for database
  • Socket.IO for real-time updates
  • Tailwind CSS for UI
  • QR Code generation with secure rotation
